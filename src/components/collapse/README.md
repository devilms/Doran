
# Collapse Attributes

参数|说明|类型|可选值|默认值
-|-|-|-|-|
value/v-model|当前激活的值|array||
accordion|是否手风琴模式|boolean|||false

# Collapse Events

事件名|说明|回调参数
-|-|-|
change|切换面板时触发，返回当前已展开的面板的name集合|array|

# Collapse-item Attributes

参数|说明|类型|可选值|默认值
-|-|-|-|-|
title|面板标题|string||
name|唯一标志符|string/number||
hideArrow|隐藏箭头|boolean||false
