# Doran-ui
基于vue的一套中后台ui框架
>注：因为没有经过系统的测试和推广，该ui目前仅供学习。

## Version
1.x

## Completed components
```$xslt
Icon (图标)
Grid (栅格)
Collapse (折叠面板)
Button (按钮)
Radio (单选)
Checkbox (多选)
Switch (开关)
Badge (微标)
Avatar (头像)
Breadcrumb (面包屑)
Tag (标签)
Progress (进度条)
Rate (评分)
Divider (分割线)
InputNumber (数字输入框)
Input (输入框)
Select (选择器)
Pagination (分页器)
```

## Demo
```
git clone
```
