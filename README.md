# doran-ui
基于vue的一套中后台ui框架
>注：因为没有经过系统的测试和推广，该ui目前仅供学习组件开发，暂时不推荐项目中使用

## version
1.x

## Completed components
```$xslt
Icon (图标)
Grid (栅格)
Collapse (折叠面板)
Button (按钮)
Radio (单选)
Checkbox (多选)
Switch (开关)
Badge (微标)
Avatar (头像)
Breadcrumb (面包屑)
Tag (标签)
Progress (进度条)
Rate (评分)
Divider (分割线)
```
